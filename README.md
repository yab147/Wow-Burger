# Wow Burger

Wow Burger is a responsive restaurant website for browsing burgers, fries, drinks, chicken meals, and desserts.

## Features

- Home, menu, item detail, and contact pages
- Search and category filters for menu items
- Light and dark mode toggle
- Mobile-friendly layout
- Local image assets, no build step required

## How to Run

Open `index.html` in your browser.

For a local server, you can also run:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000`.

## Files

- `index.html` - page structure
- `style.css` - layout, colors, and responsive styles
- `app.js` - menu rendering, navigation, search, and theme toggle
